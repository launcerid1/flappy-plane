## 🎮 Cara Bermain

1. Klik layar atau tekan **Spasi** untuk membuat pesawat terbang ke atas.
2. Hindari tabung/rintangan yang muncul dari kanan.
3. Kumpulkan skor setiap kali berhasil melewati rintangan.
4. Jika pesawat menabrak, game over — tekan **Enter** atau klik untuk restart.

---

## 🕹️ Kontrol

| Input | Aksi |
|---|---|
| **Klik / Tap** | Pesawat terbang ke atas |
| **Spasi** | Pesawat terbang ke atas |
| **Enter** | Restart setelah game over |

## 🚀 Cara Menjalankan Secara Lokal

```bash
# 1. Clone repository
git clone https://github.com/launcerid1/flappy-plane.git

# 2. Masuk ke direktori
cd flappy-plane

# 3. Jalankan server lokal Python
python server.py

# 4. Buka di browser
http://localhost:8000
```

Atau cukup buka `index.html` langsung di browser untuk mode offline.

## ✨ Fitur

- ✈️ Pesawat sebagai karakter utama
- 🌤️ Latar belakang langit dengan awan
- 🏗️ Rintangan menara (atas dan bawah)
- 📊 Sistem skor dan high score
- 🔄 Game over + restart dengan satu tombol
- 📱 Kontrol klik, tap, atau keyboard
