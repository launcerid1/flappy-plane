# Flappy Plane

Game Flappy Bird-style dengan pesawat penumpang.

## Mainkan Langsung

[https://launcerid1.github.io/flappy-plane](https://launcerid1.github.io/flappy-plane)

## Cara Menjalankan Lokal

```bash
git clone https://github.com/launcerid1/flappy-plane.git
cd flappy-plane
python server.py
```

## Fitur

- Pesawat sebagai karakter
- Rintangan menara
- Kontrol klik, tap, atau spasi
- Skor dan high score
- Game over dan restart
